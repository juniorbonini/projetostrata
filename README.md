🎨 Projeto Strata - Portfólio Pessoal
Este projeto é uma implementação de um site de portfólio responsivo, focado no domínio de técnicas avançadas de CSS puro e estruturação de layouts complexos sem o uso de frameworks externos como Bootstrap.

🛠️ Tecnologias e Conceitos Dominados
Diferente de projetos básicos, este repositório demonstra habilidades fundamentais para um desenvolvedor Web & Mobile:

Custom Grid System: Implementação manual de um sistema de colunas (row, col-2, col-1-3) utilizando float e box-sizing, demonstrando compreensão profunda de como o layout da web é construído.

Posicionamento Estratégico: Uso de position: fixed para criar um Header lateral estático enquanto o conteúdo principal (main-content) flui de forma independente, uma técnica comum em dashboards profissionais.

CSS Typography & UI:

Integração com Google Fonts e FontAwesome.

Tratamento de tabelas com nth-child(even) para Zebra Striping.

Normalização de estilos com normalize.css.

Formulários Estilizados: Criação de inputs, textareas e seletores personalizados com estados de :focus dinâmicos.

HTML5 Semântico: Uso rigoroso de tags como <header>, <main>, <section>, <article>, <address> e <footer> para melhor acessibilidade e indexação (SEO).

🚀 Diferenciais Técnicos
Manutenibilidade: Uso de seletores CSS organizados que permitem a fácil alteração da paleta de cores e tipografia.

Efeitos de Transição: Links e botões com transition suave, elevando a percepção de qualidade do usuário (UX).

Layout Adaptativo: O conteúdo principal ocupa 65% da largura, garantindo uma leitura confortável e moderna.
